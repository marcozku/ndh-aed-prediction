# 项目概述

## 项目名称
NDH AED 病人數量預測系統（North District Hospital A&E Department Attendance Forecaster）

## 项目描述
香港北區醫院急症室每日就診人數預測系統。提供 1-30 天 horizon 的預測 + 80%/95% 信賴區間 + 模型診斷儀表板。整合 GPT 主因子分析 / HKO 天氣 / CHP 流感監測 / 學校學期 / 持續學習。

## 当前版本
- **應用**: v5.5.00 (2026-05-18 HKT)
- **模型 pipeline**: `horizon_direct_xgboost` v5.5.00
- **真實 walk-forward MAE**: **13.84 人** (4186 天 honest 60/40 split, **v5.0.00 → v5.5.00 −22.9%**)
- **MAPE**: **6.04%**
- **CI80 經驗覆蓋率**: 81-83%（CQR + online conformal calibrated）
- **5 buckets**: short / h7 / h14 / h21 / h30 (per-horizon split)
- **92 features**: 含 weather + flu + school + holiday-type + AI + holiday context
- **4 base learners**: XGBoost (Optuna) + LightGBM + N-BEATS + TFT

## 技术栈

### 後端
- Node.js 18 (`server.js` 346KB)
- PostgreSQL 16 (Railway)
- Python 3.11 子進程做模型 inference 與 training
- node-cron 排程
- opossum 斷路器 + lru-cache 5-bucket cache layer

### 模型
- XGBoost (per-bucket Optuna 30-40 trials TPE 調參)
- LightGBM (L1 objective) 第二 base learner + auto-fallback blend
- N-BEATS (`neuralforecast` 731K params, identity+trend+seasonality) 全域 anchor, blend 0.15
- **TFT (Temporal Fusion Transformer)** 56K params, hidden=32, 2-head, blend 0.10
- Quantile regression (q10/q90) + Conformalized δ offsets
- Capped per-DoW bias correction + auto-fallback safety valve
- Online conformal CI calibration (predict_range 拉 30 天殘差)
- **Hierarchical Bayesian shrinkage**（軟版，h14/h21/h30 anchor 0.10）
- **HKO 9-day forecast injection** at inference (`fetch_hko_9day_forecast()` + 6h cache)

### 前端
- 純 ES module / Service Worker / PWA
- Chart.js / responsive mobile cards
- index.html footer 顯示版本 / model / GPT arm / pipeline metadata

### 外部資料
- HKO weather_history 4186 行
- CHP Flu Express 646 週 → 4515 daily rows (`python/chp_flu_express.csv`)
- HK EDB school calendar 13 年 / 54 segments (`python/hk_school_calendar.json`)
- HK 公眾假期 2014-2027
- learning_records + daily_predictions UNION 為 165 行 AI factor
- GPT-5.5 即時新聞分析

## 开发说明

### 開發/部署
- Railway 自動部署 main 分支
- `nixpacks.toml` 配置 Node 18 + Python 3.11
- `python/requirements.txt` 含 xgboost, lightgbm, neuralforecast (PyTorch), optuna

### 重要約束
- **永遠用 HK 時間（HKT）**
- **永遠用真實資料**，禁 mock
- **每次重大改動更新 `VERSION_LOG.md` + `package.json` 版本號 + `.tasks/current.md` + `algorithm_timeline.json` + 前端 footer + service worker cache version**
- **不報 R²**（walk-forward 不適用），改報 MAE/MAPE/CI coverage
- **絕對不接受 random split 算的指標**，只接受時序 walk-forward 真實 metrics
- **不在 post-hoc 加固定乘子**，所有外生訊號（天氣/AI/流感/學校）必須進 model feature
