模型置信度

📊 數據品質計算公式
總分 = 基礎 + 近期 + 覆蓋率
• 基礎分數 = min(數據天數/1000, 1) × 50
• 近期分數 = (30天內數據/30) × 30
• 覆蓋率 = 日期連續性 × 20
--%
數據品質
🤖 驗證表現計算公式
表現分數 = 0.6 × baseline skill + 0.4 × MAPE score
• baseline skill：以 MAE 相對 naive baseline 的優勢換算
• MAPE score：以近期百分比誤差分級換算
• 優先用最近已驗證資料，沒有才退回訓練摘要
• direct multi-horizon serving 不以 R² 作主要生產指標
--%
驗證表現
🎯 近期準確度計算公式
準確度 = 100% - MAPE
• MAPE = Σ|實際-預測|/實際 / N
• 基於最近 30 天比較
• 目標：準確度 > 95%
--%
近期準確度
📈 綜合置信度計算公式
綜合 = 數據×30% + 表現×40% + 準確×30%
• 驗證表現權重最高（40%）
• >90% = 高置信度 ✅
• 70-90% = 中等 ⚡
• <70% = 需審查 ⚠️
